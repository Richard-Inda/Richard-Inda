# 👋 Hi, I'm Richard!

🧙 Fantasy kid at heart.

💻 **Software Engineer** by trade.

I grew up reading stories about wizards and witches bringing fantasy to life. I found that same sense of possibility in technology. Since then, I’ve been hooked on what you can create with a few lines of code.

⚙️ I use code to automate the boring stuff, make life a little easier for people, and turn complex problems into simple solutions.

**🛠️ Currently working on:** 🏀Fantasy Helper🏀

A fantasy basketball analytics web app that pulls live NBA data and generates weekly and rest-of-season projections to help users make smarter lineup decisions. I’m rebuilding the platform from the ground up — modernizing the backend, improving data pipelines, refining projection logic, and redesigning the UI to deliver faster updates, clearer insights, and a cleaner overall experience.
